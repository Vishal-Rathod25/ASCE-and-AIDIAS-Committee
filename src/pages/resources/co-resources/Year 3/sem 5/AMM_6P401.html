Subject Description

Advanced Microprocessor & Microcontroller is applied level course aims for interfacing of various peripheral elements with microcontroller to design an automated system. This course covers introduction to families of microprocessors from 32 bit 64 bit and embed C programming environment to control peripheral devices.
